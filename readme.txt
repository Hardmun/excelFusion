1) Компонента считывает готовые файлы *.xlsx, сформированные 1С  и преобразует их в 1н файл Excel. Каждый файл в
последствии является листом. Содержания преобразуются в формулы.
2) Необхоимо указать имя временной папки.
Пример:
//1С записывает временные фременные файлы *.xlsx в каталог "UUID_Dir"
UUID_Dir = String(New UUID());
//Каталог временных файлов
bDir = Branwen.GetBranwenDirectory(UUID_Dir); - возвращает путь к временному каталогу
//Путь ко временным файлам, которые уже создал 1С
xlsx_dir = bDir + UUID_Dir + "/";

3)Пример объединения файлов, созданных в 1с

МассивФайлов = Новый Массив;

Макет = ПолучитьМакет("Макет_1");
Макет.Записать(xlsx_dir + "ИмяФайла" + ".xlsx");
МассивФайлов.Добавить("ИмяФайла");

Макет_2 = ПолучитьМакет("Макет_2");
Макет_2.Записать(xlsx_dir + "ИмяФайла_2" + ".xlsx");
МассивФайлов.Добавить("ИмяФайла_2");

ГотовыйФайлExcel = Branwen.GetCombineFile(МассивФайлов, bDir, UUID_Dir); - Возвращает готовый файл Excel (тип временное хранилище)
